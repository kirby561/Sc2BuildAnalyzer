This folder was created with the following steps:
1) Clone the s2protocol github repository at https://github.com/Blizzard/s2protocol/
	The commit was 956eebda5dad4b06021885afbe38abf6b91ae7ad
2) Run (with Python 2.7):
	python setup.py develop
3) Copy mpyq.py and mpyq.pyc to the s2protocol folder (a sub folder of the root folder that was created with the setup sript above)
